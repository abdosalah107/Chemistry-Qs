# Chemistry-Qs
general chemistry questions for high schoA lightweight, web-based quiz application designed to help studentsol students in chemical foundations. The app features dynamic question loading, LaTeX support for scientific formulas, and integrated visual aids.
## Features
### Scientific Notation: 
Uses MathJax 3 to render complex chemical formulas, units, and scientific notation (e.g., $2.1 \times 10^{4}$ or $H_2O$) professionally.
### Visual Learning: 
Support for embedded images to illustrate concepts like accuracy vs. precision, unit conversion charts, and laboratory equipment.
